# Slydes

## A Markdown Presentation Tool for Tsyne

---

# Welcome to Slydes!

This is a simple markdown-based presentation tool ported from the original Go/Fyne slydes project.

---

# Features

* Edit markdown in real-time
* See live preview
* Navigate between slides
* Full-screen presentation mode
* TOML front matter support

---

# Code Examples

You can include code blocks:

```javascript
function hello() {
  console.log("Hello, Slydes!");
}
```

---

# How to Use

1. Edit the markdown on the left
2. Use `---` to separate slides
3. Preview on the right
4. Click "Present" for full-screen mode

---

# Navigation

Use the arrow buttons or keyboard shortcuts:

* Next: Right arrow, Space, Enter
* Previous: Left arrow
* Exit: Escape (in presentation mode)
